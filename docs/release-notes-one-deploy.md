# Release Notes: `one-deploy`

Коротко и простыми словами: что поменяли в этой ветке.

## Обновление от 21.04.2026

- На странице игрока добавили отдельную верхнюю таблицу **«Зачет сезона»**.
- Таблица показывает те же зачётные соревнования, что раньше были только в тултипе у очков зачёта, но теперь в явном виде с заголовками колонок.
- Фильтр сезона в таблице берётся из того же выбранного сезона, что и на странице «Список игроков».
- Исправили риск падения интерфейса из-за нарушения порядка хуков React при переходе между состояниями загрузки и готовности на странице игрока.

## Что стало лучше

- Сайт и сервер теперь лучше работают вместе в одном месте.
- Обновления запускаются аккуратнее и не путаются, если их нажать несколько раз.
- Страницы соревнований теперь показывают результаты правильно, даже если данные приходят не напрямую, а через вложенные части.
- Подсчёт очков у игроков стал точнее и не теряет важные данные.
- На телефоне пользоваться сайтом стало удобнее.

## Про работу сайта и сервера

- Сайт и API теперь подружили между собой для Cloudflare.
- Сайт научился понимать секреты в том виде, как их даёт Cloudflare.
- Сервер теперь собирается и запускается и в обычной среде, и в Cloudflare без ошибок.
- Убрали лишнее сжатие данных, чтобы всё работало надёжнее.

## Про обновления данных

- Обновления теперь сохраняют свой статус, чтобы система понимала, что уже сделано.
- Если несколько процессов смотрят на одно и то же обновление, они больше не мешают друг другу.
- Добавили запись о прогрессе обновлений в базу.
- Страница для администраторов стала понятнее.
- Обновили логику для обновления соревнований, дивизионов, игроков, курсов и результатов.

## Про соревнования и очки

- Страница соревнований теперь не показывает пустоту там, где результаты уже есть.
- Если результаты приходят через вложенные части соревнования, сайт тоже их показывает.
- Список соревнований и страница с деталями теперь не спорят друг с другом.
- Подсчёт сезонных очков у игроков теперь берёт полные данные, а не урезанный кусок.
- Если в соревновании мало финишировавших, система всё равно считает очки по правилам, а не сбивается.

## Про телефон

- Добавили удобное боковое меню.
- Шапка сайта и навигация стали удобнее на маленьком экране.
- Страницы соревнований, курсов, игроков и карточка игрока стали лучше выглядеть на телефоне.
- Общие стили подстроили под новый вид.

## Что ещё сделали

- Убрали из ветки лишние служебные файлы, которые не нужны в проекте.
- Добавили и обновили тесты, чтобы такие ошибки ловились раньше.
